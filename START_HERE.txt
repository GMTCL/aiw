🚀 เริ่มใช้งาน 4 ขั้นตอน:

1️⃣  ติดตั้ง Python 3.12 (⚠️ อย่าใช้ 3.14!)
    → Microsoft Store: ค้นหา "Python 3.12"
    → หรือ: https://www.python.org/downloads/
    → ถ้ามี 3.14 อยู่แล้ว ต้องถอนแล้วติดตั้ง 3.12

2️⃣  ติดตั้ง Packages
    → เปิด Command Prompt ในโฟลเดอร์นี้
    → พิมพ์: pip install -r requirements.txt

3️⃣  สมัคร API (ฟรี)
    → ไปที่: https://replicate.com
    → สมัครสมาชิก → คัดลอก API Token
    → สร้างไฟล์ .env และใส่:
      REPLICATE_API_TOKEN=your_token_here

4️⃣  รันโปรแกรม
    → พิมพ์: python app.py
    → เปิดเบราว์เซอร์: http://localhost:5000
